# Introduzione

Benvenuti nella documentazione di Reticulum-Go.

Questo progetto fornisce un'implementazione basata su Go del Reticulum Network Stack, con supporto per WebAssembly nel browser.

## Caratteristiche

- **Stack Reticulum Completo**: Un'implementazione completa di Reticulum in Go.
- **Supporto WASM**: Esegui Reticulum-Go senza problemi nei browser moderni.
- **Integrazione Svelte**: Componenti e store Svelte facili da usare.
