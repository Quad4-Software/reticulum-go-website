# Введение

Добро пожаловать в документацию Reticulum-Go.

Этот проект предоставляет реализацию сетевого стека Reticulum на языке Go с поддержкой WebAssembly в браузере.

## Особенности

- **Полный стек Reticulum**: Полная реализация Reticulum на Go.
- **Поддержка WASM**: Бесшовный запуск Reticulum-Go в современных браузерах.
- **Интеграция со Svelte**: Простые в использовании компоненты и хранилища Svelte.
